Kurs Java - Praca w środowisku IntelliJ IDEA


Tworzenie projektu w IntelliJ IDEA

Uruchum IntelliJ IDEA i utwórz nowy projekt.

Utwórz nazwę projektu "javaTraining" i wskaż ścieżkę do zainstalowanego JDK.
IntelliJ powinien wykryć JDK na Twoim komputerze i wyświetlić je w rozwiniętym menu pod nazwą "Detected SDKs".
Gdyby tak się nie stało, należy wybrać "Add JDK..." i wskazać ścieżkę.
Pamietaj, by odznaczyć "Add sample code"
Resztę zaznacz zgodnie z ilustracją i kliknij przycisk "Create".

Utworzony projekt powinien wyglądać tak...

W menu po lewej stronie znajduje się struktura projektu i folder src wewnątrz niego oznaczony na niebiesko.
Jest to specjalny folder do przechowywania kodu źródłowego.

Kliknij na niego prawym przyciskiem myszy i utwórz w nim plik klasy o nazwie "Main".

Tak powinna wyglądać utworzona klasa...


Dokładne informacje o klasach, obiektach i metodach znajdują się w dalszej części kursu. Na tym etapie zwróć uwagę na fakt, że plik klasy (Main.java) musi nazywać się dokładnie tak samo jak klasa w nim zdefiniowana (public class Main).
Klasa posiada swoje ciało pomiędzy klamrami { }.

Wewnątrz ciała klasy możesz teraz utworzyć specjalną metodę, która rozpoczyna pracę programu. Jest to metoda main() i możesz utworzyć ją w następujący sposób...

Utwórz nową linię po klamrze otwierającej ciało klasy, wpisz skrót "psvm" i wybierz pierwszy wynik, aby utworzyć metodę startową programu.

Powinna ona wyglądać tak...

W tej chwili w ciele klasy Main jest metoda main(), która również posiada własne ciało pomiędzy klamrami { }. Struktura metody main() również będzie omówiona w dalszej części kursu. Wewnątrz ciała metody main możesz teraz napisac pierwszą instrukcję, którą wykona program.

Istnieje pewna tradycja, według której pierwszy napisany program to wyświetlenie napisu "Hello World !" (Witaj świecie !). Aby napisać instrukcję wyświetlającą taki napis możesz posłużyć się skrótem "sout". W ciele metody main wpisz skrót "sout" i wybierz pierwszys wynik.

W wygenerowanym poleceniu wpisz "Hello World !" we wskazanym miejscu i uruchom program, klikając miejsce oznaczone na ilustracji.

Uruchamianie programu za pomocą wiersza poleceń

Powyższy program można uruchomić bez użycia IntelliJ IDEA. Aby to zrobić wykonaj następujące czynności:

1. Utwórz na pulpicie plik tekstowy o nazwie Main.txt

2. Umieść kod programu w pliku i zapisz go: 

          
            public class Main {
              public static void main(String[] args) {
                  System.out.println("Hello World!");
              }
            }

3. Kliknij prawym przyciskiem myszy na utworzony plik na pulpicie i wybierz "właściwości". Zmień rozszerzenie pliku z Main.txt na Main.java oraz skopiuj lokalizację pliku.

4. Kliknij przycisk "Start" w lewym dolnym rogu ekranu, wpisz "cmd.exe" w pasku wyszukiwania i kliknij pierwszy wynik.

5. Sprawdź, czy JDK jest zainstalowane poprawnie, wpisując komendę "java -version" (powinieneś otrzymać komunikat o wersji java)

6. Następnie wpisz "cd" kliknij raz spację i wklej skopiowaną wczesniej ścieżkę do pulpitu. Polecenie powinno wyglądać mniej wiecej tak:
cd C:\Users\twoja nazwa użytkownika\Desktop.

7. Wpisz polecenie javac Main.java aby skompilować kod Twojego programu. Na pulpicie powinien pojawić się teraz plik Main.class
Powyższa instrukcja to właśnie przykład użycia Java Compiler'a, tłumaczącego kod źródłowy na pośredni.

8. Wpisz polecenie java Main aby uruchomić swój program. W konsoli powinien wyświetlić się napis "Hello World !".
Powyższa instrukcja to użycie Java Virtual Machine - uruchomienie kodu pośredniego.
