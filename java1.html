Kurs Java - Podstawowe informacje


Historia języka

Java to język programowania, który został opracowany przez Jamesa Goslinga i jego zespół w firmie Sun Microsystems, która obecnie należy do Oracle Corporation. Pierwsza publiczna wersja, Java 1.0, została wydana w roku 1996. Na przestrzeni lat powstały kolejne wersje języka, które wprowadziły usprawnienia, optymalizacje i nowe funkcje. Do roku 2023 powstała wersja Java 20.


Środowisko uruchomieniowe

Java jest językiem programowania, który jest niezależny od platformy, co oznacza, że programy napisane w Javie mogą być uruchamiane na różnych systemach operacyjnych, takich jak Windows, macOS i Linux. Aby uruchomić program napisany w Javie, należy pobrać i zainstalować Java Runtime Environment (JRE), czyli środowisko uruchomieniowe, które jest odpowiedzialne za interpretację i wykonanie kodu.

JRE składa się z kilku głównych komponentów:

Java Virtual Machine (JVM):
Element odpowiedzialny za interpretację kodu i przekształcenie go na instrukcje zrozumiałe dla systemu operacyjnego, na którym uruchamiany jest program. Za przekształcenie kodu odpowiada Just-in-time Compiler (JIT), który znajduje się w JVM.

Biblioteki:
zawierają takie funkcje jak obsługa wejścia/wyjścia, obsługa sieci, zarządzanie plikami, obsługa wątków, operacje matematyczne, obsługa grafiki, obsługa baz danych i wiele innych..

Inne narzędzia:
Narzędzia do zarządzania pamięcią, narzędzia diagnostyczne i narzędzia do profilowania wydajności. Pomagają one programistom w analizie i optymalizacji działania programów.


Narzędzia dla programistów

JRE służy do uruchomienia programu, natomiast Java Development Kit (JDK) to zestaw narzędzi programistycznych do tworzenia programów w języku Java. JDK zawiera wszystkie komponenty JRE i dodatkowo dostarcza narzędzi potrzebnych do tworzenia, kompilowania i debugowania programów Java.

Oto kilka głównych komponentów JDK:

Java Compiler (javac):
Jest to narzędzie do kompilacji kodu źródłowego napisanego w języku Java. Przetwarza pliki kodu źródłowego Java (pliki z rozszerzeniem .java) na pliki bajtowe (pliki z rozszerzeniem .class), które mogą być wykonywane przez JRE.

Java Runtime Environment (JRE):
Wyżej opisane środowisko uruchomieniowe.

Biblioteki:
Podobnie jak w przypadku JRE, JDK zawiera zestaw bibliotek klas, które programiści mogą wykorzystać podczas tworzenia aplikacji Java. Te biblioteki zawierają gotowe moduły, które implementują różne funkcje i usługi, takie jak obsługa sieci, obsługa plików, interfejsy użytkownika itp.
