Wprowadzenie do języka Java

Twórcy Javy i narzędzia dla programistów i użytkowników

Java to język programowania, który został opracowany przez Jamesa Goslinga i jego zespół w firmie Sun Microsystems, która obecnie należy do Oracle Corporation. Prace nad językiem rozpoczęły się w drugiej połowie lat 90. XX wieku, a pierwsza publiczna wersja, Java 1.0, została wydana w roku 1996.

Do roku 2023 powstała juz wersja Java 20 i wprowadzono wiele zmian. Na przykład przed wersją Java 11 można było pobrać i zainstalować osobno:
Java Runtime Environment (JRE) i Java Development Kit (JDK). Od wersji 11 do pobrania zostało tylko JDK, które zawiera JRE.

JDK - Narzędzia dla programistów. Posiadają JRE i kompilator Java Compiler, który tłumaczy kod źródłowy na kod pośredni.

JRE - Środowisko uruchamianiowe dla programów Java. Posiada Maszynę Wirtualną Java (JVM), która, za pomocą kompilatora Just-in-time Compiler (JIT) tłumaczy kod pośredni (bytecode) na maszynowy i uruchamia program.

Edytor tekstowy

Kod źródłowy Java można napisać w zwykłym notatniku lub zaawansowanym edytorze tekstowym. Napiszmy teraz pierwszy program w Javie z użyciem Wiersza poleceń, notatnika i zainstalowanego już JDK.

1. Utwórz na pulpicie plik tekstowy o nazwie Powitanie.java
2. Umieść poniższy kod w pliku i zapisz go (upewnij się, że plik ma rozszerzenie "Powitanie.java" zamiast "Powitanie.java.txt"): 

          
          public class Powitanie {
              public static void main(String[] args) {
                  System.out.println("Hello World !");
              }
          }

3. Kliknij prawym przyciskiem myszy na utworzony plik na pulpicie i wybierz "właściwości". Zaznacz i skopiuj lokalizację pliku.
4. Aby dostać się do wiersza poleceń, kliknij przycisk "Start" w lewym dolnym rogu ekranu, wpisz "cmd.exe" w pasku wyszukiwania i kliknij pierwszy wynik.
5. Sprawdź, czy JDK jest zainstalowane poprawnie, wpisując komendę "java -version" (powinieneś otrzymać komunikat o wersji java)
6. Następnie wpisz "cd" kliknij raz spację i wklej skopiowaną wczesniej ścieżkę do pulpitu. Powinno to wyglądać tak: cd C:\Users\twoja nazwa użytkownika\Desktop.
7. Wpisz "javac Powitanie.java" aby skompilować kod Twojego programu. Na pulpicie powinien pojawić się teraz plik Powitanie.class
8. Wpisz "java Powitanie" aby uruchomić swój program. W konsoli powinien wyświetlić się napis "Hello World !"

Jeśli wszystko poszło ok, to właśnie napisałeś działający program w notatniku. Jak już wcześniej wspomniałem, istnieją również zaawansowane edytory tekstowe, czyli inaczej IDE. Nazwa "IDE" jest skrótem od "Integrated Development Environment" (zintegrowane środowisko programistyczne). Pomagają one poprawić literówki w kodzie, podświetlają składnię, czyniąc kod czytelniejszym, a nawet generują kod za ciebie.
