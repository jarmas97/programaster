Instrukcja warunkowa if else

Instrukcja "if" sprawdza warunek umieszczony w nawiasach okrągłych. Jeśli warunek jest spełniony, wykonuje się kod w bloku po słowie kluczowym "if". W przeciwnym razie, wykonuje się kod w bloku po słowie kluczowym "else". Blok "else" jest opcjonalny. 

          
            if (warunek) {
                instrukcje do wykonania, jeśli warunek się spełni
            } else {
                instrukcje do wykonania, jeśli warunek się nie spełni
            }

W programie zdefiniowana jest zmienna "age" i przypisana jest jej wartość 15. Następnie w nawiasach instrukcji "if" operatorem porównania sprawdzany jest warunek ( czy zmienna jest mniejsza niż 18). W bloku kodu instrukcji if, pomiędzy klamrami { } znajduje instrukcja wykonana kiedy warunek się spełni, a w bloku kodu else, kiedy się nie spełni.

Wynikiem działania programu jest blok kodu instrukcji if "Użytkownik jest nieletni", ponieważ 15 jest mniejsze niż 18.


Instrukcja warunkowa switch case

Instrukcja "switch" sprawdza wartość wyrażenia. Jeśli wartość pasuje do jednego z przypadków "case", wykonuje się kod w tym przypadku. Każdy przypadek powinien zawierać instrukcję "break", która powoduje zakończenie działania switch-a. Jeśli żaden przypadek nie pasuje do wartości, wykonuje się kod w bloku "default". Blok "default" jest opcjonalny. 

          
            switch (wyrażenie) {
                case wartość1:
                    Kod do wykonania, jeśli wyrażenie jest równe wartości1
                    break;
                case wartość2:
                    Kod do wykonania, jeśli wyrażenie jest równe wartości2
                    break;
                default:
                    Kod do wykonania, jeśli wyrażenie nie pasuje do żadnej wartości
            }

Po uruchomieniu programu w konsoli wyświetli się napis "Środa", ponieważ zmienna dayOfWeek przechowuję wartość 3.

Instrukcje warunkowe stosuje się naprzemiennie w zależności od potrzeb użycia w programie.


Pętla for

Pętla "for" wykonuje się przez określoną liczbę iteracji (powtórzeń). W nawiasach okrągłych po słowie kluczowym "for" umieszcza się trzy części:
Inicjalizacja Określa początkową wartość licznika lub innych zmiennych kontrolujących pętlę.
Warunek Określa warunek, który musi być spełniony, aby pętla kontynuowała działanie.
Iteracja Definiuje, jakie zmiany mają miejsce po każdej iteracji, np. inkrementacja lub dekrementacja licznika. 

          
            for (inicjalizacja; warunek; iteracja) {
                Kod do wykonania w każdej iteracji
            }

Działanie pętli for w tym przykładzie jest następujące:

Początkowo, zmienna i jest ustawiona na 0.
Następnie, sprawdzany jest warunek i < 5. Jeśli jest spełniony, wykonuje się blok kodu wewnątrz pętli.
W tym przypadku, wyświetlana jest wartość i przy użyciu System.out.println(i).
Po zakończeniu wykonania bloku kodu, zmienna i jest zwiększana o 1 za pomocą i++ (inkrementacja).
Proces powtarza się, aż warunek i < 5 nie jest już spełniony (gdy i osiągnie wartość 5 lub większą).
Ostatecznie, na konsoli zostaną wyświetlone liczby od 0 do 4, ponieważ pętla wykonuje się 5 razy (dla wartości i od 0 do 4).


Pętla while

          
            while (warunek) {
                Kod do wykonania dopóki warunek jest spełniony
            }

Instrukcja "while" wykonuje się dopóki warunek jest spełniony. Warunek sprawdzany jest przed każdą iteracją. Jeśli warunek jest spełniony, kod w bloku {} jest wykonywany.

Przykład...

Działanie pętli while w tym przykładzie jest następujące:

Początkowo, zmienna number jest ustawiona na 0.
Następnie, sprawdzany jest warunek number < 5. Jeśli jest spełniony, wykonuje się blok kodu wewnątrz pętli.
W tym przypadku, wyświetlana jest wartość number przy użyciu System.out.println(number).
Po zakończeniu wykonania bloku kodu, zmienna number jest zwiększana o 1 za pomocą number++.
Proces powtarza się, aż warunek number < 5 nie jest już spełniony (gdy number osiągnie wartość 5 lub większą).
Ostatecznie, na konsoli zostaną wyświetlone liczby od 0 do 4, ponieważ pętla wykonuje się 5 razy (dla wartości number od 0 do 4).


Pętla do while

          
            do {
                Kod do wykonania przed sprawdzeniem warunku
            } while (warunek);

Instrukcja "do-while" jest podobna do pętli "while", ale różnica polega na tym, że warunek jest sprawdzany po wykonaniu bloku {}. Oznacza to, że kod wewnątrz bloku {} jest wykonywany przynajmniej raz, nawet jeśli warunek na początku nie jest spełniony. Jeśli warunek jest spełniony, kod w bloku {} jest kontynuowany.

Przykład...

Działanie pętli do-while w tym przykładzie jest następujące:

Początkowo, zmienna i jest ustawiona na 0.
Następnie, wykonywany jest blok kodu wewnątrz pętli, który wyświetla wartość i przy użyciu System.out.println(i).
Po zakończeniu wykonania bloku kodu, zmienna i jest zwiększana o 1 za pomocą i++.
Następnie, warunek i < 5 jest sprawdzany. Jeśli warunek jest spełniony, pętla zostaje ponownie wykonana, a proces się powtarza.
Proces kontynuuje się, dopóki warunek i < 5 jest spełniony.
Ostatecznie, na konsoli zostaną wyświetlone liczby od 0 do 4, ponieważ pętla wykonuje się 5 razy (dla wartości i od 0 do 4).


Tablice

Tablica w języku Java to struktura danych, która przechowuje kolekcję elementów tego samego typu. Dostęp do jej elementów jest możliwy za pomocą indeksów. Można ją zadeklarować na dwa sposoby: 

          
            int[] numbers;
            int numbers[];

Aby przypisać wartość tablicy, używa się słowa kluczowego new, następnie w nawiasach kwadratowych podaje się rozmiar tablicy. 

          
            numbers  = new int[5];

Przykład...

int[] numbers = new int[5]; Tworzy nową tablicę numbers o rozmiarze 5, która będzie przechowywać liczby całkowite.
Pięć kolejnych instrukcji to przypisanie różnych pięciu wartości liczbowych do każdego elementu tablicy.
Należy zwrócić uwagę na fakt, że indeksy zaczynają się od 0, a nie od 1.
Pięć następnych instrukcji to wyświetlanie każdego elementu według jego indeksu.
